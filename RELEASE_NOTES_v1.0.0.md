# v1.0.0 - MIT-licensed supplementary software release

Initial MIT-licensed release of Supplementary Software 1 for the Paper 2 manuscript.

This release contains a reviewer-runnable Python prototype for:

- synthetic-curated MFP-Score ranking;
- OPR candidate generation;
- feature-diagnostic output;
- uncertainty reporting;
- checksum and validation workflow.

This package is a reproducibility prototype and does not claim validated production ML performance, completed DFT/MD calculations, autonomous-lab execution, or experimentally verified synthesis routes.
